# Консультации по нейросетям: логика работы

Один сценарий для менеджера: **звонок → заполнение карточки в процессе → отправка клиенту готовой индивидуальной PDF-карты**.

---

## Логика работы по шагам

### 1. До консультации

- Открыть **Карточка-консультанта-нейросети** (файл `.md` или `.html`).
- Просмотреть блоки **«Во время консультации»**: вопросы по потребностям, профиль клиента (профессия, цель, уровень), таблицу «Где нейросети помогают» и подсказки по профессиям.
- Открыть в другой вкладке **Карта-пути-для-клиента.html** — её будешь показывать клиенту и потом из неё делать PDF.

### 2. Во время консультации

- **Вести разговор по карточке консультанта:** заполнять по ходу сферу, роль, процессы, что отнимает время, главную цель, уровень знакомства с нейросетями.
- **Заполнить профиль клиента** (блок «Профиль для индивидуальной карты»): профессия/роль, главная задача (заработок / оптимизация / масштаб / обучение с нуля), уровень (новичок / пробовал / использую). Отметить акцент при показе карты.
- **Показать клиенту карту** (`Карта-пути-для-клиента.html`): пройти по шагам 1–4, опираться на блоки **«Под ваш профиль»** под его профессию и цель.
- **Записать** в карточке 1–2 первые задачи, которые клиент готов начать делать с нейросетями, и следующую цель / формат следующего шага.

### 3. После консультации

- **В карточке консультанта:** дочистить заметки, заполнить «Что могу предложить я», при необходимости — заработок по услугам (см. **Заработок-по-услугам.md**). Карточка остаётся у менеджера (внутренний документ).
- **В карте для клиента** (`Карта-пути-для-клиента.html`): в блоке **«Ваш профиль»** подставить данные этого клиента (профессия, цель, уровень). При желании слегка подправить формулировки под него.
- **Сохранить карту в PDF:** в браузере открыть `Карта-пути-для-клиента.html` → Печать (Ctrl/Cmd + P) → «Сохранить как PDF».
- **Отправить клиенту** готовую индивидуальную PDF-карту и договорённости о следующих шагах.

**Итог:** клиент получает одну PDF — его персональную карту пути с его профилем и акцентами под его задачи и профессию.

---

## Как собрать карту индивидуально под клиента

1. **Открой** `Карта-пути-для-клиента.html` в браузере (один раз на консультацию — можно держать рядом с карточкой консультанта).
2. **Заполни в карте** (всё можно править прямо в браузере):
   - **Твой профиль** — профессия, цель, уровень (подставь то, что записал в карточке консультанта).
   - **Под тебя** — одна фраза-акцент под этого клиента (например: «Тебе особенно фазы 1 и 3 — тексты и визуал» или «Фокус на заработке: фазы 1, 3, 5»). Кликни по блоку и замени текст.
   - **Твои первые шаги** — 1–2 конкретных шага, которые вы договорились на консультации (например: «1. Зарегистрироваться в ChatGPT и написать первый пост. 2. На этой неделе сделать контент-план на месяц»). Кликни по блоку и впиши или замени.
3. **Сохрани в PDF:** Ctrl/Cmd + P → «Сохранить как PDF» → отправить клиенту.

Карта одна и та же для всех, но **профиль + «Под тебя» + «Первые шаги»** делают её индивидуальной: клиент видит свои данные и свои следующие действия.

### Как сделать карту по-настоящему «под него»

- **Имя** — вверху страницы есть поле с именем. Впиши имя клиента: в PDF появится строка «Мария, это твоя карта» над заголовком. Если имя не нужно — очисти поле или оставь пустым (в PDF пустая строка не печатается).
- **Под тебя** — одна фраза-акцент под его цель и профессию (например: «Тебе особенно фазы 1 и 3» или «Фокус на заработке: тексты + визуал»). Клиент видит, что карта собрана под него.
- **Первые шаги** — два шага с нумерацией и кружками: консультант редактирует текст каждого шага. Договорились на консультации — вписал в карту, клиент получает чёткий чеклист.
- **Дата** — внизу карты поле «Карта для тебя · дата». Впиши дату консультации или «март 2025» — документ выглядит персонально и актуально.

Так каждый клиент получает один PDF с **именем**, **профилем**, **акцентом под него**, **конкретными шагами** и **датой** — без отдельных шаблонов.

## Файлы в папке

| Файл | Кто использует | Зачем |
|------|----------------|-------|
| **Карточка-консультанта-нейросети.md** или **.html** | Менеджер | Заполнять до/во время/после консультации. Не отправлять клиенту. |
| **Карта-пути-для-клиента.html** | Менеджер → клиент | Показать на консультации, заполнить профиль, сохранить в PDF и отправить клиенту. |
| **Заработок-по-услугам.md** | Менеджер | Справочник: сколько зарабатываешь на каждой услуге, где смотреть цифры в компании. |
