# Заработок по услугам — где смотреть и сколько зарабатываешь на каждой

Используй этот файл как **справочник**: подставь свои услуги и актуальные цифры из компании. Тогда в консультации можно быстро прикинуть, что выгоднее предлагать.

---

## Где в компании посмотреть, сколько платят за каждую услугу

| Где искать | Что там обычно |
|------------|----------------|
| **Положение о премиях / мотивации продаж** | Процент с сделки, бонусы за объём, условия выплат по продуктам |
| **CRM** | Настройки комиссий по продуктам/услугам, типы сделок и привязка к выплатам |
| **Таблица / гугл-таблица от руководителя** | Сводка: услуга → % или фикс за единицу, пороги, градации |
| **Внутренний портал, база знаний, чат** | Объявления об изменении мотивации, памятки по расчёту |
| **Бухгалтерия / HR** | Уточнение формул расчёта, когда в документах неочевидно |
