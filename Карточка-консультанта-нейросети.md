# Карточка консультанта: нейросети и потребности клиента

**Логика:** до консультации — просмотреть блок «Во время». Во время — заполнять карточку и показывать клиенту карту. После — дочистить заметки, заполнить профиль в карте клиента, сохранить карту в PDF и отправить клиенту.

---

## До консультации

- [ ] Открыл карточку и карту для клиента (`Карта-пути-для-клиента.html`).
- [ ] Просмотрел вопросы и подсказки по профессиям ниже — готов вести разговор.

### Профиль для индивидуальной карты

*Заполни обязательно — эти данные потом подставишь в карту клиента перед сохранением в PDF.*

- **Профессия / роль:** маркетолог, SMM, бухгалтер, руководитель, фрилансер, предприниматель, HR, юрист, другое _______________________
- **Главная задача клиента:** заработок / оптимизация времени / масштаб / обучение с нуля / другое _______________________
- **Уровень:** новичок / уже пробовал нейросети / использую иногда
- **Акцент при показе карты:** _______________________

**Подсказки по профессиям:**

| Профиль | На что сделать акцент в карте |
|---------|-------------------------------|
| Маркетолог, SMM | Контент-планы, посты, визуалы, рассылки, экономия времени, масштаб без доп. штата |
| Бухгалтер, финансы | Документы, сводки, шаблоны писем и отчётов, регламенты |
| Руководитель | Делегирование рутины, выжимки из отчётов, масштаб по команде |
| Фрилансер | Заработок: больше заказов, новые услуги (контент, консалтинг) |
| Предприниматель | Оптимизация маркетинга и контента, автоматизация заявок и рассылок |
| Новичок | Старт с одной задачи, первый результат за неделю |
| Цель — заработок | На каждом шаге: сколько можно зарабатывать, как монетизировать |
| Цель — оптимизация | Часы экономии, процессы на автомате, метрики до/после |

### Где нейросети помогают (подсказки для разговора)

| Обработка заявок, писем, чатов | Классификация, ответы по шаблону, суммаризация |
| Анализ данных, отчёты | Выборка, сводки, визуализация идей |
| Обучение и онбординг | Объяснения, тесты, сценарии под роль |
| Креатив (дизайн, идеи) | Идеи, мудборды, варианты формулировок |
| Работа с документами | Поиск, краткое изложение, сравнение версий |
| Продажи и маркетинг | Персонализация, сценарии писем, скрипты |

### Отправка клиенту

1. Открыть **Карта-пути-для-клиента.html**.
2. В блоке **«Ваш профиль»** подставить данные этого клиента (профессия, цель, уровень) из карточки выше.
3. Печать → «Сохранить как PDF».
4. Отправить клиенту готовую индивидуальную PDF-карту.
